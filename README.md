## Development

1. Clone the repository:
```bash
git clone https://github.com/farisnceit/Branch-name-generator.git
cd Branch-name-generator
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Run tests:
```bash
npm test
```
